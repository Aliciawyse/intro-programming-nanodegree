Major Takeaways from Stage One

History and Evolution of HTML

HTML stands for Hypertext Markup Language. Markup Languages are not the same as programming languages, they use tags to annotate documents. HTML1 was created in 1990. In 1993, Mosaic emerged as the first graphical browser. As time went on, more browsers were created. Some had their unique tags. Eventually, web standards were developed. Today, browsers continue to adopt new standards.

How HTML Works

The internet is made up of local area networks. Multiple local area networks that are connected together are called wide area networks. There are client and server relationships on the network. Servers are machines that hold chared resources. Clients are our computers. The request response cycle is what happens when you, the client, request a page or a service from a server.

A URL has three parts: the protocol, domain and document. When you type in a URL, the browser takes that info and sends out an HTTP or HTTPS request. Typically, this happens in multiple rounds. A server will return files. A browser will decide what to do with the files.

HTML Tools and Tips

Uses dashes, not uderscores in your file names. Trouble-shooting is normal. It's often necessary to refresh your page. Create folders for your files and name your files carefully. You can use Sublime, TextEdit and Notepad (and many other text editors)to create your webpage.

Cascading Style Sheets

Webpages look plain by default, but you can add style. You can style attributes to HTML tags directly. However, it's convention to separate content from style. So, we use a Cascading Style Sheet - a way for you to write rules on how to style your html.

Typically, when you write CSS you first, identify the tag you want to style, aka your selector. Once you define what tag you want to change, you can say what about that selector you'd like to change, aka its property (color, background color). Afterwards, you can give the property a value. Altogether, you can say, whenever you see an h1 element change its color to blue. Note, the most recent rule has precedence, but htis can be overwritten if you choose.
